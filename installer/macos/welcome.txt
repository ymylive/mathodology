Welcome to Mathodology.

This installer lays down the gateway binary, web SPA, worker source, and
launchd units under /usr/local/mathodology and /Library/LaunchDaemons.

Postgres and Redis are NOT bundled — install them separately (Homebrew works
well). The postinstall script will print the exact commands.

This package is UNSIGNED (no Apple Developer ID). If the installer refuses to
open, run from Terminal:
    xattr -d com.apple.quarantine ./mathodology-*.pkg
or allow it via System Settings -> Privacy & Security.
